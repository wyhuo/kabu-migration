# BallDontLie API 注册步骤

## 快速注册指南

### 步骤 1：访问官网
打开浏览器访问：
https://www.balldontlie.io/

### 步骤 2：注册账号
1. 点击 "Sign Up" 或 "Get API Key"
2. 填写邮箱和密码
3. 验证邮箱

### 步骤 3：获取 API Key
1. 登录后进入 Dashboard
2. 复制 API Key（格式：长字符串）

### 步骤 4：配置到系统
把 API Key 发给我，我会配置到环境变量。

或者手动设置：
```bash
export BALLDONTLIE_API_KEY="your_key_here"
echo "BALLDONTLIE_API_KEY=your_key_here" >> ~/.bashrc
```
