# 球星卡收藏入门指南

## 什么是球星卡？

球星卡（Trading Cards）是一种印有运动员照片和信息的收藏品，具有观赏、收藏和投资价值。

---

## 卡片类型

### 按内容分类
| 类型 | 说明 | 价值特点 |
|------|------|----------|
| Base/普卡 | 基础卡片，数量最多 | 流通性好，增值有限 |
| Parallel/平行卡 | 普卡的特殊版本（不同颜色/折射） | 稀有度决定价值 |
| Insert/特卡 | 特殊主题设计的卡片 | 设计感强，收藏价值 |
| Auto/签字卡 | 球员亲笔签名 | 签名真实，高价值 |
| Patch/球衣卡 | 嵌入球衣布料 | 实物嵌入，独特 |
| RPA | 新秀签字+球衣 | 顶级收藏，最高价值 |
| 1/1 | 唯一一张 | 最高稀有度 |

### 按新秀分类
| 类型 | 说明 | 重要性 |
|------|------|--------|
| RC (Rookie Card) | 新秀卡 | 球员首张NBA卡，最具价值 |
| Rookie Patch Auto | 新秀球衣签字 | 新秀卡中的顶级 |

## 新手入门步骤

### 1. 确定收藏方向
- **球星选择**：喜欢哪位球员？
- **系列选择**：Prizm（投资）vs Mosaic（收藏）
- **预算范围**：决定收藏规模

### 2. 学习卡品评估
**四角 (Corners)**：卡片四个角的完整度
**边缘 (Edges)**：卡片边缘的磨损
**表面 (Surface)**：划痕、指纹、印刷缺陷
**居中 (Centering)**：图像是否居中

### 3. 了解价格参考
- eBay Sold Listings（真实成交价）
- 130point.com（综合价格搜索）
- Card Ladder（价格指数）

### 4. 开始购买
- **建议渠道**：eBay、卡淘、卡圈群
- **注意**：新手先买低价卡练手

## 保管与存储

### 基础保护
1. **卡膜 (Sleeves)**： penny sleeve，基础防护
2. **卡夹 (Toploaders)**：硬壳保护，日常展示
3. **卡砖 (One-Touch)**：磁吸硬壳，高价值卡专用

### 环境要求
- 避光存放（防止褪色）
- 恒温恒湿（防止变形）
- 远离磁场（保护签名墨迹）

### 高级存储
- 评级卡盒（PSA/BGS slab box）
- 防火保险箱（高价值收藏）

## 常见术语

| 术语 | 含义 |
|------|------|
| Box/盒 | 整盒未拆封卡包 |
| Pack/包 | 单包卡片 |
| Break/拆卡 | 直播或录制拆盒过程 |
| Hit/ hit卡 | 盒中的好卡（签字/球衣/低编）|
| RIP | 拆卡完成 |
| PC (Personal Collection) | 个人收藏 |
| FS/FT | For Sale / For Trade |
| OBO | Or Best Offer |

## 投资风险提示

### ⚠️ 高风险
1. **FOMO追高**：高光后立即买入
2. **过度集中**：所有资金押注1-2个球员
3. **忽视流动性**：买了发现卖不出去
4. **假卡风险**：高价值卡务必验证

### ✅ 降低风险
1. 分散投资多个球员
2. 关注基本面而非短期 hype
3. 只买能长期持有的卡
4. 从正规渠道购买
