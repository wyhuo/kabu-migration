## 🟢 买入决策清单

### 第一步：基础信息检查
- [ ] 球员已列入关注清单
- [ ] 卡片系列认可（Prizm/Select/Topps Chrome）
- [ ] 卡品良好（四角/边缘/表面/居中）
- [ ] 卖家信誉良好

### 第二步：数据验证
- [ ] 近期比赛数据已查看
- [ ] 数据呈上升趋势或稳定
- [ ] 上场时间充足（>20分钟）
- [ ] 无重大伤病

### 第三步：价格评估
- [ ] 对比近30天成交价格
- [ ] 当前价格在合理区间
- [ ] 未处于 hype 峰值
- [ ] 符合预算范围

### 第四步：风险评估
- [ ] 单球员持仓 < 15%
- [ ] 新秀卡总计 < 40%
- [ ] 现金储备充足
- [ ] 可承受30%回撤

### 第五步：时机判断
- [ ] 非休赛期（7-9月除外）
- [ ] 非 hype 后立即追高
- [ ] 非重要比赛前（避免买在预期）

## 🔴 卖出决策清单

### 卖出信号检查
#### 数据层面
- [ ] 数据连续下滑（3场以上）
- [ ] 上场时间大幅减少
- [ ] 重大伤病
- [ ] 球队地位下降

#### 市场层面
- [ ] 价格处于 hype 峰值
- [ ] 成交量萎缩（流动性差）
- [ ] 更好的投资机会出现

#### 基本面
- [ ] 球队交易/角色变化
- [ ] 负面新闻
- [ ] 年龄因素（老将）

#### 组合层面
- [ ] 单卡占比过高需调整
- [ ] 需要现金

## 💡 决策原则

### 买入原则
1. **数据驱动** - 不凭感觉，只看数据
2. **分散投资** - 不把鸡蛋放一个篮子
3. **分批建仓** - 不一次性满仓
4. **长期视角** - 至少持有一个赛季观察

### 卖出原则
1. **止盈不犹豫** - 达到目标果断卖出
2. **止损要坚决** - 跌破止损线立即执行
3. **不赚最后一个铜板** - 不追求卖在最高点
4. **机会成本** - 有更好的机会可以换仓

### 观望原则
1. **宁可错过不做错** - 不确定时不行动
2. **数据说话** - 等数据验证后再决策
3. **保持关注** - 观望不等于遗忘
