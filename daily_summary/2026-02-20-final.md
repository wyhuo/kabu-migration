## 四、🔌 MCP协议学习

### MCP (Model Context Protocol) 入门

**学习日期：** 2026-02-20  
**官方文档：** https://modelcontextprotocol.io

#### 1. 什么是MCP？

**Model Context Protocol (MCP)** 是一个开放协议，用于标准化 AI 模型与外部数据源、工具和服务之间的集成。它由 Anthropic 创建并开源。

**核心目标：**
- 统一 AI 模型与外部系统的连接方式
- 提供标准化的接口和协议
- 使 AI 应用能够安全地访问和操作外部资源

#### 2. MCP 架构概览

```
┌─────────────────────────────────────────────────────────────┐
│                     MCP 架构概览                             │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────────┐        MCP Protocol        ┌──────────┐  │
│  │  MCP Client  │  ◄──────────────────────►  │ MCP Server│  │
│  │  (AI 应用)    │      JSON-RPC 2.0         │ (工具/数据)│  │
│  └──────────────┘                           └──────────┘  │
│         │                                          │        │
│         │          ┌─────────────┐                 │        │
│         └────────► │  Transport  │ ◄───────────────┘        │
│                    │  (stdio/SSE)│                         │
│                    └─────────────┘                         │
└─────────────────────────────────────────────────────────────┘
```

#### 3. 核心组件

| 组件 | 描述 | 示例 |
|------|------|------|
| **Tools** | 可执行功能，修改状态 | 文件读写、API调用、命令执行 |
| **Resources** | 只读数据 | 文件内容、数据库记录、配置信息 |
| **Prompts** | 预定义提示模板 | 标准化AI交互模板 |

#### 4. 搭建MCP服务器示例

**Python实现（计算器服务器）：**
```python
from mcp.server import Server
from mcp.types import TextContent, Tool

app = Server("calculator-server")

@app.list_tools()
async def list_tools():
    return [
        Tool(
            name="add",
            description="Add two numbers",
            inputSchema={
                "type": "object",
                "properties": {
                    "a": {"type": "number"},
                    "b": {"type": "number"}
                },
                "required": ["a", "b"]
            }
        )
    ]

@app.call_tool()
async def call_tool(name: str, arguments: dict):
    if name == "add":
        result = arguments["a"] + arguments["b"]
        return [TextContent(type="text", text=str(result))]
```

#### 5. OpenClaw集成方法

**配置示例：**
```yaml
# openclaw_config.yaml
mcp:
  servers:

```
